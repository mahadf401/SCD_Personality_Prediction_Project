# AI Personality Prediction Web Application

This project is a Flask-based machine learning web application that predicts personality traits using the K-Means clustering algorithm.

## Features
- Personality prediction using machine learning
- Flask web interface
- Graphical result visualization
- Threading implementation
- Unit testing using unittest
- API testing using Postman
- Trello project management
- UML diagrams

## Technologies Used
- Python
- Flask
- Pandas
- Scikit-learn
- HTML/CSS
- Machine Learning
